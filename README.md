## 下载

### Docker

```
docker pull apachecn0/flygon-js-zh
docker run -tid -p <port>:80 apachecn0/flygon-js-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g flygon-js-zh
flygon-js-zh <port>
# 访问 http://localhost:{port} 查看文档
```
