## 下载

### Docker

```
docker pull apachecn0/freelearn-html-css-js-zh
docker run -tid -p <port>:80 apachecn0/freelearn-html-css-js-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g freelearn-html-css-js-zh
freelearn-html-css-js-zh <port>
# 访问 http://localhost:{port} 查看文档
```
