Preface

Welcome to Sencha Architect App Development. The goal of this book is to help you learn how to use Sencha Architect to boost your productivity when developing Sencha (Ext JS and Sencha Touch) applications by guiding you through a series of complete examples and tips and tricks.

Sencha Architect is an HTML 5 visual app builder that allows you to create Sencha Touch (mobile) and Ext JS (desktop) applications by dragging-and-dropping components and previewing your application live. All code built in Sencha Architect is designed to be best practice compliant, making it easy to build large applications or simply to learn. And all the code that is generated could be written manually by you, which demonstrates how powerful this tool is.

What this book covers

Chapter 1, Introducing Sencha Architect, introduces Sencha Architect and its interface features such as project basics, the toolbox, the config panel, the project inspector, and the design canvas.

Chapter 2, Creating an Ext JS App, demonstrates how to create a complete Ext JS application using the MVC architecture and Sencha Architect and also how to execute a project for the first time.

Chapter 3, Creating a Sencha Touch App, covers how to create a complete Sencha Touch application using the MVC architecture and Sencha Architect. This chapter also demonstrates how it is possible to integrate a Sencha Touch application with Phonegap.

Chapter 4, Tips and Tricks, explores some valuable tips and tricks that can help a developer on a daily basis when working with Sencha Architect. Some of the topics covered are using third-party plugins, multilingual apps, creating custom components for reuse, and much more.

Chapter 5, Working with Resources, explains how to use the resources toolbox and the resources package from the project inspector.

Chapter 6, Simulating, Building, Packaging, and Deploying a Project/App, explains how to prepare the application for production by previewing it (in the browser), packaging, building, and deploying.

What you need for this book

The following is a list of the software you will need to have installed prior to executing the examples of the book.

Use the following browser:

  • Google Chrome, which is recommended to execute Sencha Touch applications (http://www.google.com/chrome)

Use the following webserver with PHP support (or any other webserver of your preference):

  • Xampp (http://www.apachefriends.org/en/xampp.html)

Use the following database:

  • MySQL (http://dev.mysql.com/downloads/mysql/)
  • MySQL Workbench or any other database admin (http://dev.mysql.com/downloads/tools/workbench/)

Use the following Sencha tools and required complementary software:

  • Sencha Cmd (http://www.sencha.com/products/sencha-cmd/download)
  • Ruby (http://www.ruby-lang.org/en/downloads/)
  • Sass (http://sass-lang.com/)
  • Compass (http://compass-style.org/)
  • Java JDK (http://www.oracle.com/technetwork/java/javase/downloads/index.html)
  • Java environment variables (http://docs.oracle.com/javase/tutorial/essential/environment/paths.html)
  • PhoneGap (http://phonegap.com/)
  • Ext JS (http://www.sencha.com/products/extjs/)—we will use Ext JS 4.2
  • Sencha Touch (http://www.sencha.com/products/touch/)—we will use Sencha Touch 2.2
  • Sencha Architect 2.2 or higher (http://www.sencha.com/products/architect/)

Use the following iOS tools (optional – Mac OS only):

  • XCode with Command Line (https://developer.apple.com/xcode/)

Use the following Android tools:

  • Eclipse is optional, but highly recommended (http://www.eclipse.org/downloads/)
  • Android SKD (http://developer.android.com/sdk/index.html)

Who this book is for

This book is for developers who are already familiar with Ext JS and Sencha Touch (possessing at least basic knowledge). Familiarity with coding HTML, a basic background in CSS, a strong background in JavaScript, and basic understanding of JSON is also very welcome. This book is also for developers who want to learn more about Sencha Architect and how to use it to boost productivity.
